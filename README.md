### Requirements
`sqlite3`, `g++` 7 or newer, [slqite_orm](https://github.com/fnc12/sqlite_orm)

#### With nix installed
`nix-build assigment-4`
#### Without nix installed
`assignment-4/build.sh`
### Run the binary
#### If built with nix-build
`result/assignment-4`
#### If built with build.sh
`builds/assignment-4`
